Getting Started


Clone from Github

git clone https://github.com/spatial-computing/prisms-air-quality-modeling.git


Open Project in IntelliJ

Drag the PRISMS folder into IntelliJ.



Add Spark in Libraries in Project

1. Download Spark version 2.1.0 and unzip it.

2. Open the file -> Project Structure, in the intelliJ.

3. In the Libraries tab in the left, click the add button.

4. Find jar folder in your unzipped spark folder.

5. Click Done and Apply buttons.


Configurate Database Connection

Change the username and password in the ConnectDB.



Establish SSH Tunnel

Open the Terminal and Run the command

ssh -L 11223:localhost:5432 yijun@jonsnow.usc.edu

The port and the username maybe change depend on the project code.



Let's Run the Code!

Run the main function in the demo package.

We can get result at the result folder
